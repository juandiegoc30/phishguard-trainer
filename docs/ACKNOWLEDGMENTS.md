# Acknowledgments

PhishGuard Trainer is a static project with no build step and minimal dependencies. This file credits the open source work it relies on.

## Dependencies

### Tailwind CSS

- **License:** MIT
- **Source:** [tailwindcss.com](https://tailwindcss.com)
- **Usage:** Loaded from CDN (`https://cdn.tailwindcss.com`) for utility-class styling across all pages.

## Tooling (development only)

The project uses only Node.js built-ins for validation and testing (`node --check`, `node tests/scoring.test.js`). No additional npm packages are installed or bundled.

## Assets

- Hero and 404 illustrations created for this project. Not derived from third-party sources.

- Favicon is an original SVG created for this project.
